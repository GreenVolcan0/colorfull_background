## Генератор цвета фона на React

Простое приложение на React: при клике по карточке генерируется случайный цвет, отображается его hex-код, а фон всей страницы меняется на этот цвет.

### Как запустить

- **Вариант 1. Открыть напрямую**
  - Просто откройте файл `index.html` в браузере (двойной клик или через контекстное меню).

- **Вариант 2. Через локальный сервер (рекомендуется)**
  - Установите зависимости:
    - `npm install`
  - Запустите простой сервер:
    - `npm run start`
  - Откройте в браузере адрес, который покажет утилита `serve` (обычно `http://localhost:3000` или `http://localhost:5000`).

### Как это работает

- **`ColorGenerator`** — React-компонент, в котором:
  - хранится текущее значение цвета в состоянии;
  - по клику генерируется новый случайный hex-код;
  - через `useEffect` фон всего `document.body` меняется на выбранный цвет.
- Отображается:
  - превью цвета (цветной прямоугольник);
  - текущий hex-код вида `#RRGGBB`;
  - подсказка «Нажмите ещё раз для нового случайного цвета».
